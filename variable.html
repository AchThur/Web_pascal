Les variables

Les variables en pascal sont composé de trois types:
les variables de type numérique, c'est à dire les entier(integer) comme "1,2,3,4", les réels(real) comme "1.123123".
Les variables de type Caractère, les caractères(char)"a,b,c,d,e" et les chaines de caractère(string) "czdzdzzdzd"
les variables de type boolean , la on défini notre variable pour que le program nous retourne sois vrai sois faux.

attention il faut toujours déclaré ces variables sinon le program ne marchera pas voicie des exemples de comment déclarer ces variables
